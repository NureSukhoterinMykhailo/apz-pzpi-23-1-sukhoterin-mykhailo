# LifeBalance Mobile App

Android/Kotlin demo application for Laboratory Work №2.

## Features
- login screen;
- dashboard;
- habit tracking;
- goals;
- analytics;
- profile screen;
- notification and IoT sync demo actions.

## How to run
1. Open Android Studio.
2. Select **Open** and choose this project folder.
3. Wait for Gradle sync.
4. Run the `app` configuration on an emulator or Android device.

## Project structure
- `MainActivity.kt` — main application logic and UI.
- `AndroidManifest.xml` — Android configuration.
- `build.gradle` — Gradle configuration.

This is a functional educational prototype that demonstrates the required mobile part of the LifeBalance system.
