# LifeBalance WebApp LR3 Advanced

Ускладнена front-end версія системи LifeBalance для лабораторної роботи №3.

## Функції
- окремі режими User/Admin;
- локалізація UA/EN;
- RTL-напрям тексту;
- темна/світла тема;
- CRUD для звичок;
- цілі з прогресом;
- Canvas-аналітика;
- журнал подій;
- пошук, фільтрація, сортування;
- резервне копіювання, експорт та імпорт JSON;
- збереження стану у localStorage.

## Запуск
Варіант 1: відкрити `index.html` у браузері.

Варіант 2 через локальний сервер:
```bash
npx serve .
```
